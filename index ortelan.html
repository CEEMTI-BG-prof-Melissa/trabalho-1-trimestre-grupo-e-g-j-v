Sumário

Requisitos Funcionais Requisitos Não Funcionais Requisitos Inversos

Requisitos Funcionais

  1. O sistema poderia permitir que os usuários anexassem atividades complementares aos livros recomendados.
  2. O sistema deve verificar quais livros estão disponíveis e quais já foram emprestados.
  3. O sistema deve enviar notificações aos usuários sobre a data de devolução dos livros.
  4. O sistema pode mostrar recomendações aos usuários com base nos livros lidos anteriormente.
  5. O sistema deve manter um controle de empréstimos e devoluções.
  6. O sistema pode contar com opções de livros digitais (e-books).
  7. O usuário deve reservar seus exemplares com até 2 dias de antecedência.
  8. O usuário deve solicitar acesso ao sistema digital da biblioteca.

Requisitos Não Funcionais

  1. O prazo máximo de posse dos livros da biblioteca será de 20 dias.
  2. O sistema permite apenas professores anexarem atividades complementares aos livros.
  3. O sistema possibilita aos professores acompanharem o progresso dos alunos.
  4. Apenas o diretor pode ter acesso ao painel de monitoramento.
  5. O sistema permite professores indicarem leituras personalizadas para os alunos considerando suas dificuldades e interesses.

Requisitos Inversos

  1. O sistema não deve permitir que os usuários mantenham posse dos materiais por tempo indeterminado.
  2. O sistema não pode deixar de mostrar quais são os materiais mais utilizados.
  3. O sistema não pode liberar mais de um exemplar por usuário.
  4. Os livros online não podem estar disponíveis para download.
